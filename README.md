# The Alignment Constraint Framework

A structural framework on AI alignment, specification coherence, separable objective
specification, substrate-aware optimization, and the Stability Assumption.

**Central question:** Can a finite, separable objective remain coherently specified as an
optimization system becomes capable enough that acting well requires modeling the conditions
its objective excludes? The framework argues that every identified strategy for maintaining
this boundary faces a named structural pressure, and reduces the open question to specific
verification tasks.

**Proof status:** Stage 4 — candidate architecture under named premises, not theorem closure.
